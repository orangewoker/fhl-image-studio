# FHL Image Studio 方汤圆版 V2.0.1 Android Release Notes

## 版本

- Tag: `v2.0.1-android`
- Package: `top.fangtangyuan.fhlstudio.android`
- Version name: `2.0.1`
- APK SHA256: `F794064CD3704889A88FE59E5F914652E18B91C131BCE173DF3463A68619A1F6`

## 重点更新

- Android 版首次公开发布源码与 APK。
- 默认 FHL 一键配置保持 `Responses / openai / https://www.fhl.mom / gpt-5.5 / gpt-image-2`。
- 支持文生图、图生图、多参考图输入。
- final 图自动写入系统相册 `Pictures/ImageStudio`。
- 新增“保存原图”和系统分享入口。
- 新增提示词复制/清空快捷按钮。
- 画布底部工具栏新增横向滑动提示。
- 修复真机状态栏、底部手势区与页面内容重叠问题。

## 开源协议与来源

本项目是基于 `RoseKhlifa/Image-Studio` 的独立修改发行版，不是上游官方版本。

原项目地址：

```text
https://github.com/RoseKhlifa/Image-Studio
```

本仓库按 GNU AGPLv3.0 公开源码，许可证见仓库根目录 `LICENSE`。下载本 APK 的用户可以通过本 Release 对应的源码 tag/source archive 获取对应源码。

## 安全

- 不内置 API Key。
- 不携带本机历史、输入图、输出图、raw 日志或审计日志。
- Android Release 与旧桌面版 Release 分开；不会覆盖旧桌面版资产。
