# FHL Image Studio 方汤圆版 V2.0.1.1 Android

这是 FHL Image Studio 方汤圆版 Android V2.0.1.1 的 AGPL 合规发布说明。

## 来源与协议

本版本基于 [RoseKhlifa/Image-Studio](https://github.com/RoseKhlifa/Image-Studio) 深度修改，是独立发行版，不是上游官方版本。

本版本按 GNU AGPLv3.0 公开对应源码：

```text
versions/android-v2.0.1.1/source
```

## 发布资产

Android 发布资产位于：

```text
versions/android-v2.0.1.1/release-assets
```

资产包含 APK 和同名 ZIP。

## 功能摘要

- FHL 一键配置。
- 多图图生图。
- Android 相册保存。
- 保存原图和系统分享。
- 提示词复制/清空。
- 画布工具栏滑动提示。

## 安全说明

- 不内置 API Key。
- 首次运行需要用户自行配置 FHL API。
- APK 和 ZIP 不包含本机私有配置、生成图、输入图、运行日志或审计日志。
