# CHANGELOG

## Desktop V2.0.1 - 2026-06-07

本版本为桌面版封板发布，用于公开上传 GitHub 和创建 `desktop-v2.0.1` Release。

### Added

- 顶部开源致谢条，明确展示 Image-Studio 原作者 RoseKhlifa 项目地址和方汤圆修改版项目地址。
- 提示词模块重排：提示词标题外置，主提示词、复制、清空、模板/历史、AI 优化、反推提示词、指令改写和风格模板统一归属同一模块。
- `指令改写提示词`：把用户输入的修改方向作为必须执行的改写指令，用于精准修改当前 prompt。
- `反推提示词`：支持导入/拖入反推专用图片，上传到支持视觉输入的文本模型后生成中文文生图 prompt。
- 主提示词和指令输入框自动增高，长文本尽量完整显示。
- 出图张数选项选中态蓝框提示。

### Changed

- 顶部 `AI 优化` 只做基础系统提示词润色，不再混用下方精准修改内容。
- 下方 `精准修改` 只在填写指令改写内容时可用，便于连续微调。
- 复制、清空按钮改为有边界的紧凑按钮，放在主提示词框右上方。
- 反推按钮移动到反推图像输入框内部，只有导入图片后显示。
- GitHub、反馈和 Star 入口统一指向方汤圆修改版仓库。
- 桌面版版本号统一为 `2.0.1 / V2.0.1`。

### Fixed

- 修复 prompt 优化方向为空时不应改变原版基础系统提示词的问题。
- 修复 prompt 优化返回解析兼容性，降低 `Upstream did not return a usable prompt optimization result` 出现概率。
- 修复反推提示词返回解析兼容性，支持多种 Responses API 文本输出结构。
- 修复反推图片入口不直观、无图也显示反推按钮的问题。

### Compliance

- 新增根目录 `README.md`、`COMPLIANCE.md`、`NOTICE.md`、`RELEASE_NOTES_DESKTOP_V2.0.1.md`。
- 新增发布合规扫描脚本。
- 发布源码排除 API Key、本机配置、测试图、生成图、日志、缓存和构建产物。

## Desktop V2.0.0

- 初始桌面修改版发布，包含 FHL API 配置、文生图/图生图、历史记录、画布和多平台前端基础能力。
