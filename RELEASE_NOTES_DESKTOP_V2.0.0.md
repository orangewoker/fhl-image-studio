# FHL Image Studio 方汤圆 CLI 魔改版 V2.0.0 Desktop

这是 FHL Image Studio 方汤圆版桌面 V2.0.0 的 AGPL 合规发布说明。

## 来源与协议

本版本基于 [RoseKhlifa/Image-Studio](https://github.com/RoseKhlifa/Image-Studio) 深度修改，是独立发行版，不是上游官方版本。

本版本按 GNU AGPLv3.0 公开对应源码：

```text
versions/desktop-v2.0.0/source
```

## 发布资产

桌面发布资产位于：

```text
versions/desktop-v2.0.0/release-assets
```

其中包含重新整理的干净桌面便携包和对应 ZIP。

## 安全说明

- 不内置 API Key。
- 首次运行需要用户在 UI 中自行配置 FHL API。
- 发布包不包含本机私有配置、运行日志、输入图、输出图或中间图。
